## Courses

### [Get started with Jira](https://www.coursera.org/projects/get-started-with-jira)

Completed course and certificate. A .md file contains further explanation what I learned in this course, which covers the following sections:
- Basic knowledge about Jira and Agile methodology,
- How to create a project and choose framework,
- How to set up a Kanban board,
- How to create issues,
- How to set WIP limits,
- How to create Epics,
- How to manage workflow and transitions,
- How to invite team members,
- How to build and manage the Timeline.
